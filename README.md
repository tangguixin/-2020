### 功能设计
    第一是推荐，各种组装机的搭配方案，按照所需推荐适合的单子.
    第二是论坛，主要是个人的发表，可发图发视频，主要是围绕自己装机的想法和一些问题展开，他人可以回复.
    第三是个人的界面，完善个人信息，查看自己收藏的浏览内容或发表方案和论坛，反馈问题.
    第四是电脑诊断，主要简单用于电脑故障排除和常见电脑问题搜索解决方案.
